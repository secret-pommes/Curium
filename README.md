# Curium

A Fortnite backend made for Nintendo Switch (should work for PC)
<br>
<br>
Codebase is written in JavaScript (node.js)
<br>
<br>

### Supported Versions:

-Season 4 - 5
<br>
-Season 6 - 9 (but with not player pawn in lobby)

### Features:

-Go into the Lobby
<br>
-Account System (WIP)

### Planned:

-Cloudstorage
<br>
-ClientSettings
<br>
-XMPP
<br>
-Matchmaker
<br>
-Queue
<br>
-Battle Pass and leveling in the pass.
<br><br>

### GameServer-Part:

-Quests
<br>
-Item Shop
